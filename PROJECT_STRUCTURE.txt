MÓDULOS PRINCIPALES
==================

1. AUTENTICACIÓN (3 archivos)
   - Supabase Auth con email/password
   - Middleware para renovación de sesiones
   - Logout seguro

2. BASE DE DATOS (2 archivos SQL)
   - Tablas: students, student_activities, student_documents
   - Índices, triggers, RLS policies

3. API REST (5 endpoints)
   - /api/students (CRUD)
   - /api/dashboard/summary
   - /api/reports/summary
   - /api/reports/export

4. COMPONENTES UI (4 componentes principales)
   - StudentForm (formulario CRUD)
   - StudentList (tabla de listado)
   - ReportGenerator (generador reportes)
   - Sidebar (navegación)

5. SERVICIOS (2 servicios)
   - StudentService (lógica de estudiantes)
   - ReportService (lógica de reportes)

6. UTILIDADES (3 módulos)
   - Validaciones
   - Seguridad
   - Manejo de errores

7. PÁGINAS (8 páginas)
   - Auth Login
   - Dashboard
   - Students (listado)
   - Create Student
   - Edit Student
   - Reports
   - Otros

TECNOLOGÍAS UTILIZADAS
====================

Frontend:
  - Next.js 16 (Full-stack React framework)
  - React 19 (UI library)
  - TypeScript (Type safety)
  - Tailwind CSS v4 (Styling)
  - Shadcn/ui (UI components)
  - Lucide React (Icons)
  - SWR (Data fetching)

Backend:
  - Next.js API Routes
  - TypeScript
  - Supabase SDK

Base de Datos:
  - PostgreSQL
  - Supabase (Backend-as-a-Service)
  - Row Level Security

Herramientas:
  - pnpm (Package manager)
  - ESLint (Code quality)
  - Vercel (Hosting)
  - GitHub (Version control)

CÓMO NAVEGAR EL CÓDIGO
====================

1. Para entender el flujo:
   - Lee app/page.tsx → middleware.ts → app/auth/login/page.tsx
   - Lee app/dashboard/page.tsx → components/layout/sidebar.tsx
   - Lee app/dashboard/students/page.tsx → components/students/student-list.tsx

2. Para entender el CRUD:
   - Lee lib/services/student-service.ts
   - Lee app/api/students/route.ts
   - Lee lib/hooks/use-students.ts
   - Lee components/students/student-form.tsx

3. Para entender la base de datos:
   - Lee scripts/01_create_tables.sql
   - Lee lib/types/student.ts

4. Para entender validaciones:
   - Lee lib/utils/validation.ts
   - Lee lib/utils/security.ts
   - Lee lib/services/student-service.ts
